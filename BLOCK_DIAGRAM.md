# Блок-схема алгоритма голосового ассистента "Лера"

## Общая архитектура

```
┌─────────────────────────────────────────────────────────────────┐
│                     ИНИЦИАЛИЗАЦИЯ                                │
├─────────────────────────────────────────────────────────────────┤
│  1. Загрузка Vosk модели (распознавание речи)                   │
│  2. Создание очередей: data_queue, tts_queue                     │
│  3. Инициализация LeraBrain (логика ассистента)                  │
│     - Настройка Tuya устройства (умная розетка)                  │
│     - Инициализация таймеров и будильников                       │
│  4. Запуск FastPiper (синтез речи в фоне)                       │
│     - Запуск piper процесса                                      │
│     - Запуск aplay процесса для воспроизведения                  │
│  5. Запуск фонового потока background_tasks_checker             │
└─────────────────────────────────────────────────────────────────┘
                              │

┌─────────────────────────────────────────────────────────────────┐
│                    ГЛАВНЫЙ ЦИКЛ                                  │

│    │                                                             │

│    │   ├─► Есть сообщение в tts_queue?                           │
│    │   │   ├─ ДА:                                                │

│    │   │   │   └─ Запустить поток микрофона                     │
│    │   │   └─ НЕТ: продолжить                                   │
│    │   │                                                         │
│    └─► СЛУШАНИЕ МИКРОФОНА                                       │
│        │                                                         │
│        ├─► Получить аудио данные из data_queue                   │
│        │                                                         │

│        │   │   │   └─ ДА:                                        │

│        │   │   │       └─ Запустить поток микрофона             │
│        │   │   │                                                 │
```

## Детальная блок-схема: LeraBrain.handle()

```
┌─────────────────────────────────────────────────────────────────┐
│              brain.handle(text) - Обработка команды              │
├─────────────────────────────────────────────────────────────────┤
│  ВХОД: распознанный текст                                       │
│                                                                 │
│  1. Привести текст к нижнему регистру и удалить пробелы        │
│                                                                 │
│  2. ПРОВЕРКА: listening_mode == False?                          │
│     ├─ ДА:                                                       │
│     │   ├─ ПРОВЕРКА: имя "лера" в тексте?                       │
│     │   │   ├─ ДА:                                               │
│     │   │   │   ├─ listening_mode = True                         │
│     │   │   │   └─ ВЕРНУТЬ: "Да, слушаю."                       │
│     │   │   └─ НЕТ:                                              │
│     │   │       └─ ВЕРНУТЬ: None                                │
│     │                                                           │
│     └─ НЕТ: (режим прослушивания активирован)                    │
│         ├─ listening_mode = False                               │
│         │                                                       │
│         ├─ ПРОВЕРКА: "будильник" + ("выключи"/"отмени"/...)     │
│         │   └─ ДА: cancel_alarms()                              │
│         │                                                       │
│         ├─ ПРОВЕРКА: "таймер" + ("выключи"/"отмени"/...)         │
│         │   └─ ДА: cancel_timer(text)                           │
│         │                                                       │
│         ├─ ПРОВЕРКА: "погода" в тексте                          │
│         │   └─ ДА: get_weather()                                │
│         │                                                       │
│         ├─ ПРОВЕРКА: "новости"/"новость" в тексте               │
│         │   └─ ДА: get_news()                                   │
│         │                                                       │
│         ├─ ПРОВЕРКА: "время"/"час"/"времени" в тексте           │
│         │   └─ ДА: get_time()                                  │
│         │                                                       │
│         ├─ ПРОВЕРКА: "таймер" в тексте                          │
│         │   └─ ДА:                                              │
│         │       ├─ words_to_numbers(text) → получить числа      │
│         │       ├─ Есть числа?                                   │
│         │       │   └─ ДА: start_timer(число)                   │
│         │       │   └─ НЕТ: "На сколько минут поставить таймер?"│
│         │                                                       │
│         ├─ ПРОВЕРКА: "будильник" в тексте                       │
│         │   └─ ДА:                                              │
│         │       ├─ words_to_numbers(text) → получить числа      │
│         │       ├─ Есть числа?                                   │
│         │       │   └─ ДА: set_alarm(время из чисел)            │
│         │                                                       │
│         ├─ ПРОВЕРКА: "свет"/"ламп"/"розетк" в тексте            │
│         │   └─ ДА:                                              │
│         │       ├─ ПРОВЕРКА: plug настроен?                     │
│         │       │   └─ НЕТ: "Module ne nastroen"               │
│         │       │                                               │
│         │       ├─ ПРОВЕРКА: "включи" в тексте                  │
│         │       │   └─ ДА: plug.turn_on() → "свет включен"      │
│         │       │                                               │
│         │       ├─ ПРОВЕРКА: "выключи" в тексте                 │
│         │       │   └─ ДА: plug.turn_off() → "Свет выключен"     │
│         │                                                       │
│         └─ ВЕРНУТЬ: "Не поняла команду."                        │
└─────────────────────────────────────────────────────────────────┘
```

## Детальная блок-схема: synth_and_say()

```
┌─────────────────────────────────────────────────────────────────┐
│              synth_and_say(text) - Синтез речи                   │
├─────────────────────────────────────────────────────────────────┤
│  ВХОД: текст для озвучивания                                    │
│                                                                 │
│  1. ПРОВЕРКА: текст пустой?                                     │
│     └─ ДА: ВЕРНУТЬ (выход)                                      │
│                                                                 │
│  2. ОБРАБОТКА ТЕКСТА:                                           │
│     ├─ Заменить цифры на слова (num2words)                      │
│     └─ Заменить "-" на " минус "                               │
│                                                                 │
│  3. ВЫВОД: "Лера говорит: {text}"                              │
│                                                                 │
│  4. СИНТЕЗ РЕЧИ:                                                │
│     ├─ voice.speak(text) → отправка в FastPiper                 │
│     │   (текст отправляется в фоновый процесс piper)           │
│     │                                                         │
│     ├─ РАСЧЁТ ВРЕМЕНИ ВОСПРОИЗВЕДЕНИЯ:                          │
│     │   estimated_duration = len(text) / 12.0                   │
│     │   (примерно 12 символов в секунду)                        │
│     │                                                         │
│     └─ PAUZA: time.sleep(estimated_duration + 0.5)             │
│         (ожидание завершения воспроизведения)                    │
│                                                                 │
│  5. ПРОВЕРКА ОШИБОК:                                            │
│     └─ При ошибке: вывести "Ошибка Piper"                       │
└─────────────────────────────────────────────────────────────────┘
```

## Детальная блок-схема: FastPiper

```
┌─────────────────────────────────────────────────────────────────┐
│                   FastPiper - Быстрый синтез                     │
├─────────────────────────────────────────────────────────────────┤
│  ИНИЦИАЛИЗАЦИЯ (__init__):                                      │
│                                                                 │
│  1. ЗАПУСК PIPER ПРОЦЕССА:                                      │
│     subprocess.Popen([                                          │
│       "piper/piper",                                             │
│       "--model", model_path,                                     │
│       "--config", config_path,                                   │
│       "--length_scale", "1.1",                                   │
│       "--output_raw"                                             │
│     ], stdin=PIPE, stdout=PIPE, stderr=DEVNULL)                 │
│                                                                 │
│  2. ЗАПУСК APLAY ПРОЦЕССА:                                      │
│     subprocess.Popen([                                          │
│       "aplay", "-r", "22050", "-f", "S16_LE",                   │
│       "-t", "raw", "-c", "1", "-q"                              │
│     ], stdin=piper_proc.stdout, stderr=DEVNULL)                 │
│     (пайпинг вывода piper напрямую в aplay)                     │
│                                                                 │
│  МЕТОД speak(text):                                              │
│                                                                 │
│  1. Очистить текст (заменить \n на пробел)                      │
│                                                                 │
│  2. Отправить текст в piper_proc.stdin:                         │
│     piper_proc.stdin.write((text + "\n").encode('utf-8'))       │
│     piper_proc.stdin.flush()                                    │
│                                                                 │
│  (Аудио автоматически передаётся в aplay и воспроизводится)     │
└─────────────────────────────────────────────────────────────────┘
```

## Детальная блок-схема: background_tasks_checker()

```
┌─────────────────────────────────────────────────────────────────┐
│         background_tasks_checker() - Фоновые задачи             │
├─────────────────────────────────────────────────────────────────┤
│  WHILE True:                                                     │
│                                                                 │
│  1. Получить текущее время: now_str = HH:MM                     │
│                                                                 │
│  2. ПРОВЕРКА: now_str в списке alarms?                          │
│     ├─ ДА:                                                       │
│     │   ├─ Отправить сообщение в tts_queue                       │
│     │   │   "Будильник на {now_str}!"                           │
│     │   ├─ Удалить время из списка alarms                        │
│     │   └─ (Сообщение будет обработано в главном цикле)         │
│     │                                                           │
│     └─ НЕТ: продолжить                                          │
│                                                                 │
│  3. PAUZA: time.sleep(30)                                       │
│     (проверка каждые 30 секунд)                                 │
└─────────────────────────────────────────────────────────────────┘
```

## Детальная блок-схема: words_to_numbers()

```
┌─────────────────────────────────────────────────────────────────┐
│         words_to_numbers(text) - Текст в числа                  │
├─────────────────────────────────────────────────────────────────┤
│  ВХОД: текст с числовыми словами ("пять минут")                 │
│                                                                 │
│  1. Словарь числовых слов:                                      │
│     {'ноль':0, 'один':1, ..., 'пятьдесят':50}                  │
│                                                                 │
│  2. Разбить текст на слова                                      │
│                                                                 │
│  3. FOR каждое слово в тексте:                                  │
│     ├─ Слово в словаре?                                         │
│     │   ├─ ДА:                                                  │
│     │   │   ├─ Первый раз? → current = значение                 │
│     │   │   ├─ current >= 20 AND значение < 10?                 │
│     │   │   │   └─ ДА: current += значение (сложение)          │
│     │   │   └─ НЕТ:                                             │
│     │   │       ├─ Сохранить current в nums                     │
│     │   │       └─ current = значение                          │
│     │   │                                                       │
│     │   └─ НЕТ:                                                 │
│     │       ├─ Было в режиме парсинга?                          │
│     │       │   └─ ДА:                                          │
│     │       │       ├─ Сохранить current в nums                 │
│     │       │       └─ Сбросить флаг парсинга                   │
│                                                                 │
│  4. Если в режиме парсинга: сохранить последний current        │
│                                                                 │
│  ВЫХОД: список чисел [5, 30]                                    │
└─────────────────────────────────────────────────────────────────┘
```

## Детальная блок-схема: Таймеры

```
┌─────────────────────────────────────────────────────────────────┐
│                    УПРАВЛЕНИЕ ТАЙМЕРАМИ                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  start_timer(minutes):                                           │
│  ├─ ПРОВЕРКА: таймер на это время уже есть?                    │
│  │   └─ ДА: отменить старый таймер                             │
│  ├─ Создать threading.Timer(minutes * 60, _timer_end, [minutes])│
│  ├─ Сохранить в self.timers[minutes] = timer                   │
│  ├─ Запустить timer.start()                                    │
│  └─ ВЕРНУТЬ: "Таймер на {minutes} минут запущен."               │
│                                                                 │
│  _timer_end(minutes):                                            │
│  ├─ Отправить в tts_queue: "Таймер на {minutes} минут завершён!" │
│  └─ Удалить из self.timers                                      │
│                                                                 │
│  cancel_timer(text):                                             │
│  ├─ words_to_numbers(text) → получить числа                   │
│  ├─ Есть числа?                                                 │
│  │   └─ ДА:                                                     │
│  │       ├─ Таймер на это время существует?                     │
│  │       │   └─ ДА: отменить и удалить → "Таймер отменён"      │
│  │       │   └─ НЕТ: "Таймер не найден"                         │
│  │   └─ НЕТ:                                                    │
│  │       ├─ 1 таймер? → отменить его                            │
│  │       ├─ >1 таймер? → "Уточните время"                       │
│  │       └─ 0 таймеров? → "Активных таймеров нет"              │
└─────────────────────────────────────────────────────────────────┘
```

## Ключевые особенности архитектуры

1. **Асинхронность**: Фоновый процесс Piper всегда запущен и готов к мгновенному синтезу
2. **Очереди**: Используются очереди для безопасной передачи данных между потоками
3. **Состояние**: listening_mode отслеживает, ожидается ли команда после активации
4. **Многопоточность**: Отдельный поток для проверки будильников каждые 30 секунд
5. **Пайпинг**: Вывод Piper напрямую передаётся в aplay без промежуточных файлов
